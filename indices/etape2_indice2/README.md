Le carré de 5x5 est petit et donc peut être initialisé explicitement :
```
table = [
    ['*','*','*','*','*',],
    ['*','*','*','*','*',],
    ['*','*','*','*','*',],
    ['*','*','*','*','*',],
    ['*','*','*','*','*',],
]
```
Il faut maintenant le remplir avec 'a', 'b', 'c', etc, soit grâce à du code Python, soit en modifiant à la main la table.
